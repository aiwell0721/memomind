# MemoMind 产品文档

> 最后更新：2026-05-01

---

## 概述

MemoMind 是基于 SQLite 的团队知识库与智能笔记系统。它提供笔记管理、全文搜索、语义搜索、知识图谱、RAG 问答、版本历史、多工作区协作等功能。
