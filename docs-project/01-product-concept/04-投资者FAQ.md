---

### Q2: "活知识系统"具体指什么？

**A:** 

传统笔记是"死"的——写完就放在那里。活知识系统意味着：

1. **自动关联** — AI 自动发现笔记间的隐性关联
2. **知识演化** — 当新笔记创建时，自动更新相关旧笔记的上下文
3. **使用即更新** — 每次查询都在优化知识表示
4. **智能蒸馏** — 自动提取核心洞察，生成知识摘要

这就像从静态图书馆变成了有管理员主动帮你整理的知识助手。

---

### Q3: MCP 协议是什么？为什么重要？

**A:**

MCP (Model Context Protocol) 是 Anthropic 推出的开放标准，让 AI Agent 能够标准化地访问外部数据源。

**重要性：**
- 用户可以用任何支持 MCP 的 Agent (Claude、Cursor、GitHub Copilot 等) 访问 MemoMind
- 形成网络效应：Agent 越多 → 用户越多 → Agent 越多
- 建立生态壁垒，竞争对手难以复制

### Q8: AI 成本会不会吃掉利润？

**A:**

**成本结构：**
- OpenAI API: $0.01-0.03/次调用
- 平均用户每月 100 次调用 = $1-3/月
- Pro 订阅 ¥28/月 ≈ $4，毛利率 25-50%

**优化策略：**
1. **本地嵌入** — 向量生成本地完成，零成本
2. **缓存策略** — 重复查询零成本
3. **分层定价** — Pro+ 用户补贴免费用户
4. **多 Provider** — 接入国产模型降低成本

**长期趋势：** AI 成本每年下降 50%+，毛利率将持续改善
