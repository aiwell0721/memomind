## 2. 竞品详细分析

### 2.1 Obsidian

#### 基本信息

| 项目 | 内容 |
|------|------|
| **官网** | https://obsidian.md |
| **发布时间** | 2020年 |
| **用户规模** | 5000万+ |
| **定价** | 个人免费，商业 $50/年 |
| **技术** | Electron + 本地 Markdown |

#### 功能分析

| 功能 | 支持度 | 评价 |
|------|--------|------|
| Markdown 编辑 | ⭐⭐⭐⭐⭐ | 业界最佳 |
| 双向链接 | ⭐⭐⭐⭐⭐ | 首创 [[ ]] 语法 |
| 知识图谱 | ⭐⭐⭐⭐ | 功能完善 |
| 本地存储 | ⭐⭐⭐⭐⭐ | 纯本地优先 |
| 全文搜索 | ⭐⭐⭐⭐ | 基于文件名和内容 |
| 插件系统 | ⭐⭐⭐⭐⭐ | 1000+ 插件 |
| 主题定制 | ⭐⭐⭐⭐⭐ | 高度可定制 |
| 移动端 | ⭐⭐⭐ | 功能较弱 |
| AI 集成 | ⭐ | 无原生 AI |
| API 开放 | ⭐⭐ | 有限插件 API |

#### 优势

1. **本地优先** - 数据完全由用户掌控
2. **双向链接** - 开创了现代笔记链接范式
3. **插件生态** - 极其丰富的插件市场
4. **知识图谱** - 可视化知识关联
5. **Markdown 原生** - 纯文本，可移植性强

#### 劣势

1. **无原生 AI** - AI 功能依赖插件
2. **协作困难** - 非为协作设计
3. **移动端弱** - 移动端体验一般
4. **学习曲线** - 功能复杂，上手较难

#### 技术架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        Obsidian 架构                            │
├─────────────────────────────────────────────────────────────────┤
│  Frontend: Electron + React + CodeMirror                        │
│  Storage: Local Markdown files                                  │
│  Index: SQLite (metadata)                                       │
│  Plugin: JavaScript API                                         │
└─────────────────────────────────────────────────────────────────┘
```

3. **开放 API 笔记工具** - 开发者友好，需求增长
4. **本地优先 + AI** - 结合两者优势的独特产品
