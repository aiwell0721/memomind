## 项目概览

### 产品名称
MindWeaver / 思维编织者（暂定）

### MVP目标
构建一个本地优先、AI原生的笔记工具，具备基础笔记管理、AI搜索和API能力，验证产品核心价值。

### 目标用户
- 技术型知识工作者
- AI早期采用者
- 开发者

## 验收标准

### 功能验收
- [ ] 可以正常创建、编辑、删除笔记
- [ ] 支持Markdown基础语法
- [ ] 语义搜索可用且准确
- [ ] AI问答能基于笔记内容回答
- [ ] API可以正常调用
- [ ] 跨平台运行正常

### 性能验收
- [ ] 启动时间 < 3秒
- [ ] 搜索响应 < 200ms
- [ ] AI响应 < 3秒
- [ ] 内存占用 < 500MB

### 质量验收
- [ ] 无P0级Bug
- [ ] 核心功能测试覆盖 > 80%
- [ ] 代码审查通过
- [ ] 安全审查通过
